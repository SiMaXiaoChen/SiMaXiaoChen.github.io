clone方法

创建并且返回这个对象的一个副本，注意副本和原来的对象地址不是指向同一个，而且该类一定要实现Cloneable接口，这个接口是标记接口，子类而且要重写object的clone方法，告诉我们实现该接口就可以实现该类对象的克隆。

cloneable接口是一个空接口，只是起到标记作用
